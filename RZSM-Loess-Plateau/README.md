This repository contains the code and data for the study of 
"A Machine Learning Framework for Estimating Multi-Layer Root Zone Soil Moisture on the Loess Plateau Using Bias-Corrected ERA5-Land Reanalysis Data"

## Contents
- `data/` — In-situ TDR soil moisture observations and parameters at Wugong, Chunhua, and Yulin
- `notebooks/` — Jupyter notebooks reproducing all ML model analyses in the paper
- `scripts/` — Python scripts for bias correction, ML model training, and evaluation

## Requirements
See `requirements.txt`. Install with:
pip install -r requirements.txt

## Usage
Run notebooks in order (01 → 04) to reproduce all results.
