This repository contains the code and data for the study:
"A Machine Learning Framework for Estimating Multi-Layer Root Zone Soil Moisture on the Loess Plateau Using Bias-Corrected ERA5-Land Reanalysis Data"

## Contents
- `data/` — In-situ TDR soil moisture observations at Wugong, Chunhua, and Yulin
- `notebooks/` — Jupyter notebooks reproducing all analyses in the paper
- `scripts/` — Python scripts for bias correction, ML model training, and evaluation
